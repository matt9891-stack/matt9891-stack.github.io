# Importing Libraries

    from dash import Dash, dcc, html, Input, Output
    import plotly.graph_objects as go

## This section imports the required libraries:

Dash is used to build interactive web applications.
dcc (Dash Core Components) provides interactive elements like dropdowns and graphs.
html allows creation of HTML elements within the app.
Input and Output are used to define callbacks (interactive behavior).
Plotly Graph Objects (go) is used to create customizable visualisations.

# Creating the Dash App

    app = Dash(__name__)

This initialises the Dash application. The __name__ argument helps Dash locate resources like CSS files.

# Defining the Layout

    app.layout = html.Div([
        html.H4('Interactive color selection with simple Dash example'),
        html.P("Select color:"),
        dcc.Dropdown(
            id="dropdown",
            options=['Gold', 'MediumTurquoise', 'LightGreen'],
            value='Gold',
            clearable=False,
        ),
        dcc.Graph(id="graph"),
    ])

This defines the structure of the web app:

html.Div acts as a container for all elements.
html.H4 displays a title.
html.P adds a short instruction.
dcc.Dropdown creates a dropdown menu:
id is used to link it to callbacks.
options defines available colors.
value sets the default selection.
clearable=False prevents removing the selection.
dcc.Graph is the placeholder for the interactive chart.

## Callback Function (Interactivity)

    @app.callback(
        Output("graph", "figure"),
        Input("dropdown", "value")
    )

This defines a callback, which updates the graph whenever the dropdown value changes:

The output is the figure displayed in the graph.
The input is the selected color from the dropdown.

# Updating the Graph

    def display_color(color):
        fig = go.Figure(
            data=go.Bar(
                y=[2, 3, 1],  # Example data
                marker_color=color
            )
        )
        return fig

This function:

Receives the selected color from the dropdown.
Creates a bar chart using Plotly.
Applies the selected color to the bars.
Returns the updated figure to be displayed.

# Running the App

    app.run(debug=True)

This starts the Dash server:

debug=True enables automatic updates and error messages during development.

## Simple Plotly Express Bar Chart

    import plotly.express as px
    
    fig = px.bar(x=["a", "b", "c"], y=[1, 3, 2])
    fig.show()

Uses Plotly Express, a high-level API for quick plotting.
Creates a simple bar chart.
fig.show() renders the chart in a browser or notebook.

# Converting to FigureWidget

    import plotly.graph_objects as go
    
    fig_widget = go.FigureWidget(fig)
    fig_widget

Converts a Plotly figure into a FigureWidget.
This allows interactivity inside Jupyter notebooks (e.g., dynamic updates).

# Creating an Empty Plotly Figure

    import plotly.graph_objects as go
    
    fig = go.Figure()
    # fig.add_trace(...)
    # fig.update_layout(...)

Initialises an empty figure.
You can manually add data (add_trace) and customise the layout (update_layout).
Useful for advanced/custom visualisations.

# Embedding Plotly in Dash

    from dash import Dash, dcc, html
    
    app = Dash()
    
    app.layout = html.Div([
        dcc.Graph(figure=fig)
    ])
    
    app.run(debug=True, use_reloader=False)

This example shows:

How to embed a static Plotly figure inside a Dash app.
dcc.Graph directly displays the figure.
use_reloader=False prevents the app from running twice in some environments.

# In Conclusion

Dash enables interactive web applications using Python only.
Plotly provides both:
High-level API (plotly.express) for quick charts
Low-level API (graph_objects) for full control
Callbacks are the core of interactivity in Dash.
Combining Dash and Plotly allows building fully interactive dashboards.
